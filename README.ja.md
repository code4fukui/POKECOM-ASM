# POKECOM-ASM

Z80互換のポケットコンピュータエミュレータプロジェクト。

## 機能

- Z80互換CPU
- 8KB RAM
- 24桁3行のディスプレイ（5x7ドット文字）

## アセンブラ

本プロジェクトには、マシン語プログラムを作成するための組み込みアセンブラが含まれています。主な機能は以下の通りです：

- 基本的なアセンブリ言語構文のサポート（例: ラベル、定数、ディレクティブ）
- コードをアセンブルしてエミュレータのメモリにロードする機能
- アセンブリ言語からBIOSルーチンを呼び出すサンプルの提供

## BIOS

エミュレータには、様々なユーティリティ関数を備えたBIOSが組み込まれています：

- ブレークポイントの登録とクリア
- メモリの読み書き
- テキストとグラフィックスの表示
- シリアルポートへのアクセス

## 使い方

1. モニタモードに入る（例: BASICモードから）
2. `USER`コマンドでマシン語用のメモリ領域を確保する
3. テキストエディタでアセンブリコードを編集する
4. `A`コマンドでコードをアセンブルする
5. `G`コマンドでコードを実行する
