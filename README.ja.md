# POKECOM-ASM

PC-G803は SHARP のポケットコンピュータのモデルです。Z80互換CPUを搭載し、8KB RAMを備えた24桁3行表示のコンパクトなデバイスです。

## 機能

- Z80互換CPU
- 8KB RAM
- 24桁3行、5x7ドット表示
- 機械語モニタ機能
- アセンブラ機能
- BIOSコールによる表示制御
- 11ピンI/Oポートによる入出力制御

## 使い方

1. 機械語モニタで、`USER0200`コマンドで機械語領域を確保します。
2. テキストエディタでアセンブリ言語のプログラムを作成します。
3. アセンブラモードで`A`コマンドを実行し、アセンブルします。
4. 機械語モニタで`G100`コマンドを実行し、アセンブルしたプログラムを実行できます。
